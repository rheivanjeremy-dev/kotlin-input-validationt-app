# 📱 Kotlin Calculator with Input Validation

A simple Android calculator app built using Kotlin. This app allows users to input two numbers, perform basic arithmetic operations, and includes input validation to ensure a smooth and safe user experience.

## ✨ Features
- Input 2 numbers
- Input validation (prevent empty fields)
- Toast notification for invalid input
- Basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Handles division by zero safely
- Displays results instantly

## 🧠 How It Works
1. User enters two numbers
2. App checks if input fields are empty
3. If empty → shows error message & Toast
4. If valid → performs calculations:
   - Addition
   - Subtraction
   - Multiplication
   - Division
5. Results are displayed on screen

## 🛠️ Tech Stack
- Kotlin
- Android Studio
- XML Layout
