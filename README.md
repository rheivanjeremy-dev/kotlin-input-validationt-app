# 📱 SmartCalc Android

A simple Android calculator app built using Kotlin. This app demonstrates basic arithmetic operations along with input validation to ensure a smooth user experience.

## ✨ Features
- Input validation (prevent empty input)
- Addition, subtraction, multiplication, division
- Handles division by zero
- Toast notification for user feedback
- Clean and simple UI

## 🛠️ Tech Stack
- Kotlin
- Android Studio
- XML Layout

## 📸 Preview
- Input two numbers
- Click "Hitung"
- See results instantly

## 🚀 How It Works
1. User inputs two numbers
2. App validates input
3. Performs calculations:
   - Addition
   - Subtraction
   - Multiplication
   - Division
4. Displays results on screen

## ⚠️ Notes
- Division by zero is handled safely
- Toast message appears if input is empty
