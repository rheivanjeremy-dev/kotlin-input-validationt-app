# 📱 Kotlin Calculator App

A simple Android app built with Kotlin to input two numbers, perform basic arithmetic operations, and display the results with input validation.

## ✨ Features
* Input numbers (2 fields)
* Basic arithmetic operations:
  * Addition
  * Subtraction
  * Multiplication
  * Division
* Input validation (prevent empty input)
* Toast notification for errors
* Real-time result display
* Handles division by zero

## 🧠 How It Works
1. User enters two numbers
2. App validates the input fields
3. If input is empty → shows error message
4. If valid → performs calculations
5. Results are displayed instantly

## 🛠️ Tech Stack
* Kotlin
* Android Studio
* XML Layout
